## 1. 设计规范速查（内嵌版，权威版见 design.longzhuge.md）

### 1.1 全局 Token
```css
--paper:         #FFFFFF;  /* 白场底，禁止米色/奶油色 */
--ink:           #1A1A1A;  /* 正文墨黑 */
--ink-soft:      #606060;  /* 次级文字 */
--camel-gold:    #BE8F5B;  /* 驼金：CTA按钮平涂、包装场景 */
--imperial-gold: #B99359;  /* 御金：图标、描边、点缀，面积≤5% */
--seal-red:      #9E2B22;  /* 印章红：价格、确认类按钮 */
```

### 1.3 排印
- 大标题/页面标题/大号价格数字：**思源宋体**（Noto Serif SC / Source Han Serif SC），正文思源黑体
- 系列名可用宽字距（letter-spacing 0.4–0.5em），但必须确认容器不折行、不溢出
- section 标题用「」直角引号包裹
- 价格格式统一：`¥:5980 元`
- 正文 ≥14px 才允许宋体，≤12px 一律黑体

### 1.4 形制与节奏
- 圆角 0–2px，禁止大圆角胶囊（既有胶囊组件可暂保留形状但必须平涂去渐变）
- 禁止金色渐变，一律平涂
- 暗场（沉浸展示区）与白场（信息区）滚动交替，不允许连续两屏暗场
- 满版图上的 overlay 文字必须有 0→40% 底部压暗渐变保证可读性
- 暗场按钮：直角 + 1px 御金描边 + 透明底；白场主按钮：墨黑底白字或印章红底白字

### 1.5 大气编辑排版（视频参考风格，与上述规则叠加执行）
- **去框化**：卡片无描边、无投影，分组靠留白与对齐；弱分组暗示只允许底色差 ≤3%
  （纯白上放 #FBFAF7）。分割线原则不用，必须用时 ≤1px 且不深于 #F1EEE8。
  输入框用下划线式或 1px 淡暖灰 #E8E4DC，禁止冷灰 #DDDDDD。
- **尺度对比**：页面主标题宋体 40–44px（@2x），说明文字 20–24px，标题:正文字号比 ≥1.8:1，
  中间不设过渡层级；金额/积分大数字宋体可到 56–64px。
- **图占屏**：banner/系列入口/商品主图通栏出血（或全局统一左右 ≤24px 留白）；
  overlay 文字带 0→40% 底部压暗渐变；单屏 ≤3 个信息组，列表每屏 ≤2.5 个卡片单元。
- **弱化 chrome**：投影仅浮层允许且 opacity ≤0.06；导航纯白无线条、标题宋体居中；
  每屏装饰元素至多 1 个（印章/竖排标签/「」标题 三选一）。
- **例外优先级**：可读性 > 大气。对比度不足、点击目标 <44pt、信息过度稀释时回退基础规范。

### 1.6 禁止项（走查重点）
微信红 #E64340 系、荧光色、蓝/冷色（青花系列主题色除外）、米色底
#F5E4C6/#FAF9F6 系、金色渐变按钮、促销感贴纸、大圆角卡片、accent 装饰条、
卡片描边盒子、冷灰 #DDDDDD 描边、页面内投影、深色分割线。

## 2. 状态文件协议（每轮必须遵守）

- `fix_plan.md` —— 任务清单，checkbox 格式。**不存在则本轮只做一件事：生成它**
  （扫描全部页面/组件，按 §1 规范逐项列出违规点，按 P0/P1/P2 排序，然后退出本轮）。
- `ralph_log.md` —— 每轮结束追加一条：日期 | 完成任务 | 改动文件 | 验证结果。
- `AGENT_NOTES.md` —— 发现的坑、页面与组件的映射关系、下轮注意事项。

## 3. 每轮工作流（严格按顺序）

1. 读 `fix_plan.md`、`ralph_log.md`、`AGENT_NOTES.md`，理解当前进度。
2. 从 fix_plan 中选 **优先级最高的一项未完成任务**（只选一项）。
3. 实施修改（仅样式/布局/UI）。
4. 验证（§4），不通过就修到通过或回滚并在 AGENT_NOTES 记录阻塞原因。
5. 勾掉任务、更新三个状态文件。
6. `git add -A && git commit -m "style(ralph): <任务描述>"`。
7. 退出。

## 4. 验证手段（按项目实际可用性选择）

- 构建必须通过：`npm run build`（或本项目实际构建命令，首轮写入 AGENT_NOTES）。
- 有 dev server 时：启动后用 Playwright/puppeteer 截图改动页面，
  对照 §1 检查色值、字体、圆角；截图存 `qa_shots/<页面>_<轮次>.png`。
- 小程序项目无浏览器可截图时：静态断言兜底——grep 全项目确认禁止色值
  （#E64340、#F5E4C6、linear-gradient.*gold 等）计数为 0，token 引用正确。
- 对比走查：每项修复在 ralph_log 中写明「修改前色值/样式 → 修改后」。
